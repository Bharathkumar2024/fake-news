## 🔄 System Flow

```
User Types News Article
        ↓
(Browser sends to API)
        ↓
🌍 Language Detector
   - Detects: English/Tamil/Hindi
   - Translates to English if needed
        ↓
🧹 Text Processor
   - Removes URLs, emails, special chars
   - Removes stopwords
   - Normalizes whitespace
        ↓
🤖 Embedding Generator
   - Converts text to 384-dim vector
   - Uses BERT language model
        ↓
🔎 FAISS Vector Search
   - Finds 5 most similar facts
   - From knowledge database
        ↓
🧠 ML Model Prediction
   - Logistic Regression classifier
   - Outputs: Fake or Real
   - With confidence score
        ↓
📊 Results Display
   - Shows prediction
   - Shows confidence
   - Shows similar facts
   - Beautiful visualization
```

## 📱 Integration Examples
