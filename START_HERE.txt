================================================================================
                    🆘 COMMON PROBLEMS & FIXES
================================================================================

PROBLEM: "Python not found"
FIX:     Install Python from https://www.python.org/downloads/
         Make sure to check "Add Python to PATH"

PROBLEM: Slow on first run (10-20 minutes)
FIX:     This is NORMAL! Training takes time
         Second run is instant. Be patient! ☕

PROBLEM: Port 5000 already in use
FIX:     Edit config.py and change FLASK_PORT = 5001

PROBLEM: Module not found errors
FIX:     Run: pip install -r requirements.txt

PROBLEM: Browser doesn't open automatically
FIX:     Open manually: http://localhost:5000

PROBLEM: Can't find terminal or command prompt
FIX:     Windows: Press Win+R, type "cmd", hit Enter
         Mac: Press Cmd+Space, type "terminal", hit Enter
         Linux: Right-click desktop, select "Terminal"
