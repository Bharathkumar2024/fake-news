### Step 2: Train Model (1 hour)

```bash

# → Browser opens automatically
```

## 📞 Troubleshooting

**"No module found"**
→ Run: `python notebooks/Step_0_Install.py`

**"CSV file not found"**
→ Place Fake.csv and True.csv in `/data` folder

**"Port already in use"**
→ Use different port in config.py

**"Out of memory"**
→ Reduce SAMPLE_SIZE in config.py

**"CUDA not available"**
→ Normal - uses CPU (slower but works fine)
